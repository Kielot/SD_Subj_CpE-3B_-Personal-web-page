My Projects

Project Description School Year

Walkie-Talkie 		An electronics project involving a basic AM transmitter and receiver circuit for communication experiments. 2nd Year - 2nd Sem

7 Segment Display 		An electronic device used to show numbers using seven small LED segments arranged in a figure-eight pattern. By turning specific segments on or off, it can display digits from 0 to 9. It is commonly used in digital clocks, calculators, and electronic meters. 3rd Year - 1st Sem (Midterms)

24 Seconds Shot Clock 		An electronic timing system designed to simulate the countdown used in basketball games. It displays a 24-second timer using a 7-segment display, which automatically counts down once activated. The system includes control buttons for start, stop, and reset functions, along with a buzzer that sounds when the time expires, indicating a violation. This project demonstrates the practical application of digital electronics, timing circuits, and basic programming or logic design in creating a functional and interactive sports device. 3rd Year - 1st Sem (Finals)
